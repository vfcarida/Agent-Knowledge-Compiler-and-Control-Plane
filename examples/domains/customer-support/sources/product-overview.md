# Product Overview and Tier Features

This document provides a quick reference for the features included in each of our subscription tiers. Agents should use this to answer "Does my plan include X?" questions.

## 1. Standard Tier (Free)

Target audience: Individual users and hobbyists.

- Core software access (web only)
- Community support forums
- 5GB Storage
- 1 Active Project

## 2. Premium Tier ($49.99/mo)

Target audience: Professionals and small teams.

- Desktop application access (Windows/Mac)
- Email and Chat support (Standard SLA)
- 100GB Storage
- Unlimited Projects
- Analytics Add-on (Available for extra $10/mo)

## 3. Enterprise Tier (Custom Pricing)

Target audience: Large organizations.

- All Premium features
- Dedicated Account Manager
- Custom SLAs with financial penalties
- SSO (Single Sign-On) Integration
- Advanced Audit Logs
- Unlimited Storage

## Add-ons

Add-ons are billed separately from the main subscription. If a customer cancels their main subscription, add-ons MUST be cancelled manually by the customer, otherwise they will continue to renew.

- **Analytics Add-on:** Advanced reporting. $10/mo (Premium only, included in Enterprise).
- **Extra Storage:** +500GB for $15/mo.
